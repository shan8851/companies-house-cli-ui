# 🏛️ companies-house-cli-ui

Landing page for [@shan8851/companies-house-cli](https://www.npmjs.com/package/@shan8851/companies-house-cli) — UK company data in your terminal.

**🌐 [ch-cli.xyz](https://ch-cli.xyz)** · **📦 [npm](https://www.npmjs.com/package/@shan8851/companies-house-cli)** · **💻 [CLI source](https://github.com/shan8851/companies-house-cli)**

## Tech

Vite + React 19 + TypeScript + Tailwind CSS v4 + Framer Motion. Dark theme, terminal examples, static SPA.

## Local dev

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # output → dist/
```
